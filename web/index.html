AI connection

Zariya works offline with a lightweight built-in assistant. Optionally connect your own Anthropic API key for full model responses — it's stored only in this browser and sent directly to Anthropic, nowhere else.

If replies fail after saving a key, your browser is likely blocking calls from a local file (CORS). Serve this folder with python3 -m http.server and open it via http://localhost instead of double-clicking it.
